## 3. 컴포넌트 상세

### 3.1 AppBar

| 속성 | 값 |
|------|-----|
| 높이 | 56px |
| 좌측 | 뒤로 아이콘 (arrow_back) |
| 타이틀 | "커뮤니티 신고 관리", fontSize 18, fontWeight 500 |
| 하단 선 | `$--border`, 0.5px |

### 3.2 필터 탭 (Pencil 디자인)

| 탭 | 배경 (활성) | 텍스트 (활성) | 배경 (비활성) | 텍스트 (비활성) |
|----|------------|--------------|-------------|---------------|
| 전체 | `$--primary` | white, w600 | `$--surface-variant` | `$--text-secondary` |
| 대기중 | 동일 | 동일 | 동일 | 동일 |
| 처리완료 | 동일 | 동일 | 동일 | 동일 |

- pill 모양: cornerRadius 20, padding [6, 14]
- fontSize 13

### 3.3 신고 카드 (_ReportCard)

| 속성 | 값 |
|------|-----|
| 패딩 | 14px |
| 배경 | colorScheme.surface |
| 모서리 | cornerRadius 12 |
| 테두리 | colorScheme.outlineVariant, 1px |
| 뱃지 (대기중) | #F59E0B26 bg, #F59E0B text, labelSmall w600 |
| 뱃지 (처리완료) | #DCFCE7 bg, #16A34A text |
| 날짜 | bodySmall, colorScheme.outline |
| 유형 | titleSmall, w600 |
| 사유 | bodySmall, 2줄 말줄임 |
| 탭 동작 | 바텀시트 모달 열기 |

### 3.4 신고 처리 바텀시트

| 속성 | 값 |
|------|-----|
| 모서리 | top cornerRadius 20 |
| 핸들 바 | 40x4, cornerRadius 2, colorScheme.outline |
| 제목 | "신고 처리", titleMedium w700 |
| 정보 영역 | surfaceContainerHighest, cornerRadius 12, padding 14 |
| 정보 행 | 라벨 52px (bodySmall w600 outline) + 값 (bodySmall) |
| 기각 버튼 | OutlinedButton, 48px, cornerRadius 12 |
| 삭제 버튼 | FilledButton, red, 48px, cornerRadius 12 |

---

## 4. 인터랙션

| 이벤트 | 동작 |
|--------|------|
| 카드 탭 | 바텀시트 모달 열기 |
| 기각 | updateStatus(dismissed) → 목록 갱신 → 토스트 |
| 삭제 및 제재 | 콘텐츠 삭제 + 알림 발송 + updateStatus(resolved) → 갱신 → 토스트 |

---

## 5. 에러/로딩 상태

| 상태 | 표시 |
|------|------|
| 로딩 | LoadingIndicator |
| 에러 | ErrorView (메시지 + 재시도) |
| 빈 목록 | EmptyState (report_off_outlined, "대기 중인 신고가 없습니다") |
