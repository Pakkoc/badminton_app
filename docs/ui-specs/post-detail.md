# 게시글 상세 — UI 화면 스펙

> 최종 수정일: 2026-02-27

---

## 1. 화면 개요

| 항목 | 내용 |
|------|------|
| **화면 ID** | `customer-post-detail` |
| **화면 명** | 게시글 상세 |
| **목적** | 공지사항 또는 이벤트 게시글의 전문을 확인한다 |
| **사용자 역할** | customer |
| **진입 조건** | 로그인 필수, `post_id` 필수 전달 |

---

## 2. 레이아웃 구조

```
┌─────────────────────────────┐

│                             │
│  스크롤 콘텐츠                 │
│  ┌─────────────────────────┐│
│  │  공지사항                 ││  ← 카테고리 뱃지
│  │                         ││
│  │  7월 휴무 안내            ││  ← 제목 (20px Bold)

│  │                         ││
```

### 섹션 설명

| 섹션 | 설명 | 비고 |
|------|------|------|
| 상단 앱바 | 뒤로가기 + "게시글" 타이틀 | 배경 `#FFFFFF`, 하단 테두리 `#E2E8F0` 0.5px |
| 스크롤 콘텐츠 | 게시글 전문 표시 | 하단 네비게이션 없음 (상세 화면) |

## 3. 컴포넌트 목록

### 3.1 상단 앱바

| 컴포넌트 | 타입 | 설명 | 스타일 |
|----------|------|------|--------|
| 뒤로가기 | IconButton | 이전 화면으로 복귀 | Material Icon `arrow_back_ios`, 24px, `#0F172A` |
| 타이틀 | Text | "게시글" | `titleLarge` (20sp, Bold), `#0F172A` |

### 3.2 게시글 콘텐츠

| 컴포넌트 | 타입 | 설명 | 스타일 |
|----------|------|------|--------|
| 카테고리 뱃지 | Chip | "공지사항" 또는 "이벤트" | pill shape, 공지: 배경 `#DCFCE7` 텍스트 `#166534`, 이벤트: 배경 `#FEF3C7` 텍스트 `#92400E` |
| 제목 | Text | 게시글 제목 | `headlineSmall` (20sp, Bold), `#0F172A` |
| 메타 정보 Row | Row | 작성자 · 날짜 | `bodySmall` (12sp), `#94A3B8` |
| 구분선 | Divider | 메타와 본문 구분 | `#E2E8F0`, 1px |
| 본문 | Text | 게시글 내용 | `bodyMedium` (14sp), `#0F172A`, lineHeight 1.6 |
| 이미지 | Image | 첨부 이미지 (있을 경우) | cornerRadius 8px, width 100% (fill_container), `BoxFit.cover` |

**콘텐츠 레이아웃:**
- 패딩: [20, 16, 32, 16] (top, right, bottom, left)
- 뱃지 ~ 제목: 12px
- 제목 ~ 메타: 8px
- 메타 ~ 구분선: 16px
- 구분선 ~ 본문: 16px
- 본문 ~ 이미지: 16px

## 4. 필드 정의

| 필드명 | 타입 | 데이터 출처 | 표시 형식 | 설명 |
|--------|------|-------------|-----------|------|
| category | Enum | posts.category | 뱃지 | `notice` / `event` |
| title | String | posts.title | 텍스트 | 게시글 제목 |
| content | String | posts.content | 텍스트 (줄바꿈 유지) | 게시글 본문 |
| author | String | "관리자" (고정) | 텍스트 | 작성자 |
| created_at | Timestamp | posts.created_at | "YYYY-MM-DD" 형식 | 작성일 |
| image_url | String? | posts.image_url | 이미지 | 첨부 이미지 URL |

## 9. 비고

- 하단 네비게이션 바를 표시하지 않는다 (상세 화면).
- 이미지가 없는 게시글은 이미지 영역을 표시하지 않는다.
- 본문 텍스트의 줄바꿈(`\n`)을 유지하여 원래 작성된 형태 그대로 표시한다.

### Flutter 구현 참고
- `StatefulWidget` (로딩/에러 상태 관리).
- 이미지 뷰어는 `photo_view` 패키지 또는 `InteractiveViewer` 위젯 사용.
