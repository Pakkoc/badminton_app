# 게시글 작성 — UI 화면 스펙

> 최종 수정일: 2026-02-20

---

## 1. 화면 개요

| 항목 | 내용 |
|------|------|
| **화면 ID** | `owner-post-create` |
| **화면 명** | 게시글 작성 |
| **목적** | 사장님이 공지사항 또는 이벤트 게시글을 작성하여 샵 상세 화면에서 고객에게 정보를 전달한다 |
| **사용자 역할** | shop_owner |
| **진입 조건** | 로그인 필수, role이 `shop_owner`, 샵 등록 완료 상태 |

### 섹션 설명

| 섹션 | 설명 | 비고 |
|------|------|------|
| 상단 앱바 | 뒤로가기 버튼 + "게시글 작성" 타이틀 | 배경 `#FFFFFF` |
| 카테고리 선택 | 공지사항 / 이벤트 선택 | 토글 형태, 반드시 하나 선택 |
| 제목 입력 | 게시글 제목 (필수) | 텍스트 필드 |

| 등록 버튼 | 게시글 등록 CTA 버튼 | 필수 필드 완료 시 활성화 |

## 6. API 연동

### 6.1 데이터 조회

이 화면에서 초기 데이터 조회는 발생하지 않는다.

### 6.2 데이터 변경

| API | 테이블/서비스 | 동작 | 파라미터 | 호출 시점 |
|-----|--------|------|----------|-----------|
| 이미지 업로드 | Supabase Storage | UPLOAD | 이미지 파일 (최대 5개) | 등록 버튼 탭 시 (게시글 INSERT 전) |
| 게시글 등록 | `posts` | INSERT | `{shop_id, category, title, content, images, event_start_date, event_end_date}` | 이미지 업로드 완료 후 |

**Supabase 쿼리 예시:**

```dart
// 1. 이미지 업로드 (있는 경우)
final imageUrls = <String>[];
for (final image in images) {
  final path = 'posts/${myShopId}/${DateTime.now().millisecondsSinceEpoch}_${image.name}';
  await supabase.storage.from('post-images').upload(path, image);
  final url = supabase.storage.from('post-images').getPublicUrl(path);
  imageUrls.add(url);
}

// 2. 게시글 등록
await supabase.from('posts').insert({
  'shop_id': myShopId,
  'category': category,  // 'notice' 또는 'event'
  'title': title,
  'content': content,
  'images': imageUrls.isNotEmpty ? imageUrls : null,
  'event_start_date': category == 'event' ? eventStartDate.toIso8601String() : null,
  'event_end_date': category == 'event' ? eventEndDate.toIso8601String() : null,
  'created_at': DateTime.now().toIso8601String(),
});
```

### 6.3 실시간 구독

이 화면에서 실시간 구독은 사용하지 않는다.

## 9. 비고

- **카테고리 선택 필수**: 카테고리(공지사항/이벤트)는 반드시 하나를 선택해야 게시글을 등록할 수 있다.
- **이벤트 기간**: 이벤트 카테고리 선택 시에만 기간 입력 영역이 표시된다. 기간은 시작일과 종료일로 구성되며, 종료일은 시작일 이후여야 한다.
- **이미지 첨부**: 최대 5장까지 첨부 가능하며, 각 이미지는 10MB 이하여야 한다. 카메라 촬영 또는 갤러리에서 선택 가능하다.
- **이미지 압축**: 업로드 전 이미지를 적절한 크기로 압축하여 저장 용량을 절약한다.
- **키보드 처리**: `SingleChildScrollView`로 감싸서 키보드가 올라올 때 스크롤 가능하게 한다.
- **작성 중 이탈 방지**: 입력 내용이 있는 상태에서 뒤로가기 시 확인 다이얼로그를 표시한다.
