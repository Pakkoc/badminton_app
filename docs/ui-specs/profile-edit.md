## 1. 화면 개요

| 항목 | 내용 |
|------|------|
| **화면 ID** | `customer-profile-edit` |
| **화면 명** | 프로필 수정 |

## 2. 레이아웃 구조

```
┌─────────────────────────────┐
│  상단 앱바                    │  56px
│  ← 뒤로가기  ·  "프로필 수정"   │
├─────────────────────────────┤
│                             │
│  스크롤 콘텐츠                 │
│                             │
│         ┌──────┐            │
│         │ 👤   │            │  ← 프로필 이미지 (80x80)

│         └──────┘            │
│                             │
│  이름                       │
│  ┌─────────────────────────┐│
│  │  홍길동                  ││  ← 입력 필드 (흰색 배경)

│                             │
│  이메일                      │
│  ┌─────────────────────────┐│

│  ┌─────────────────────────┐│

│                             │
├─────────────────────────────┤
│  ┌─────────────────────────┐│
```

### 섹션 설명

| 섹션 | 설명 | 비고 |
|------|------|------|
| 상단 앱바 | 뒤로가기 + "프로필 수정" 타이틀 | 배경 `transparent`, 하단 테두리 `#FFFFFF20` 0.5px |
| 프로필 이미지 | 아바타 이미지 + 카메라 버튼 | 중앙 정렬, 80x80px |
| 입력 필드 영역 | 이름, 이메일(읽기전용), 연락처 | 세로 정렬, gap 16px |
| 하단 저장 버튼 | "저장" 버튼 | 하단 고정, 배경 `#F59E0B` |

## 3. 컴포넌트 목록

### 3.1 상단 앱바

| 컴포넌트 | 타입 | 설명 | 스타일 |
|----------|------|------|--------|

| 타이틀 | Text | "프로필 수정" | `titleLarge` (20sp, Bold), `#FFFFFFEE` |

### 3.2 프로필 이미지

| 컴포넌트 | 타입 | 설명 | 스타일 |
|----------|------|------|--------|
| 아바타 컨테이너 | Container | 프로필 이미지 표시 | 80x80px, 원형 (cornerRadius 40px), 배경 `#FFFFFF15` |
| 기본 아이콘 | Icon | 이미지 없을 때 | `person`, 40px, `#F59E0B` |
| 카메라 버튼 | IconButton | 이미지 변경 | 28x28px, 원형, 배경 `#FFFFFF18`, 아이콘 `photo_camera` 16px `#FFFFFFCC`, 아바타 우하단에 오버레이 |

**레이아웃:**
- 중앙 정렬
- 앱바 아래 간격: 24px
- 이미지 ~ 첫 입력 필드: 24px

### 3.3 입력 필드

| 컴포넌트 | 타입 | 설명 | 스타일 |
|----------|------|------|--------|
| 이름 라벨 | Text | "이름" | `bodySmall` (12sp), `#64748B` |
| 이름 입력 | TextField | 현재 이름 표시/수정 | 배경 `#FFFFFF15`, 테두리 `#FFFFFF20` 1px, cornerRadius 14px, 높이 48px |
| 이메일 라벨 | Text | "이메일" | `bodySmall` (12sp), `#64748B` |
| 이메일 표시 | TextField (읽기전용) | 이메일 표시 (수정 불가) | 배경 `#FFFFFF15`, 테두리 없음, cornerRadius 14px, 높이 48px, 텍스트 `#FFFFFFAA` |
| 연락처 라벨 | Text | "연락처" | `bodySmall` (12sp), `#64748B` |
| 연락처 입력 | TextField | 현재 연락처 표시/수정 | 배경 `#FFFFFF15`, 테두리 `#FFFFFF20` 1px, cornerRadius 14px, 높이 48px |

**입력 필드 공통:**
- 내부 패딩: 좌우 16px, 상하 14px
- 라벨 ~ 필드: 6px
- 필드 간 간격: 16px
- 포커스 테두리: `#F59E0B` 2px
- 입력 텍스트: `bodyMedium` (14sp), `#FFFFFFEE`

### 3.4 저장 버튼

| 컴포넌트 | 타입 | 설명 | 스타일 |
|----------|------|------|--------|

**레이아웃:**

### 6.1 데이터 조회

## 9. 비고

- 이메일은 소셜 로그인에서 가져온 값이므로 **수정 불가** (읽기 전용, 회색 배경으로 시각적 구분).
- 프로필 이미지 변경 시 Supabase Storage에 업로드 후 URL을 저장한다.
- 변경사항이 있는 상태에서 뒤로가기 시 **확인 다이얼로그**를 표시한다.
- 하단 네비게이션 바를 표시하지 않는다 (상세 화면).

### 접근성
- 카메라 버튼에 `semanticLabel: "프로필 이미지 변경"`을 포함한다.
- 읽기 전용 이메일 필드에 `semanticLabel: "이메일 (수정 불가)"`를 포함한다.

### Flutter 구현 참고
- `StatefulWidget`으로 구현 (폼 상태 관리).
- `Form` + `GlobalKey<FormState>`로 유효성 검증.
- 이미지 선택은 `image_picker` 패키지 사용.
- 연락처 필드에 `keyboardType: TextInputType.phone` 적용.
