# 거트알림 - 서비스 소개

## 한 줄 요약

배드민턴 거트 작업의 접수부터 완료까지 실시간 진행 현황을 알려주어, 고객의 헛걸음을 방지하고 샵의 고객 관리를 돕는 모바일 앱.

---

## 어떤 문제를 해결하나요?

배드민턴 거트(스트링) 교체를 위해 샵에 라켓을 맡기면, "언제까지 해드릴게요"라는 구두 약속만 이루어진다. 실제 완료 시점을 정확히 알 수 없어 다음과 같은 문제가 반복된다.

- 약속 시간에 맞춰 갔지만 작업이 아직 완료되지 않아 **헛걸음하는 사례가 빈번**하다.
- 고객은 작업 진행 상황을 알 수 없어 **불안감과 불편함**을 느낀다.
- 샵은 고객 문의 전화를 반복적으로 받는 **운영 비효율**이 발생한다.

거트알림은 이 간극을 메운다.

- 사장님이 작업 상태를 변경하면 고객에게 **푸시 알림이 즉시 전송**된다.
- 고객은 앱에서 **작업 진행 현황을 실시간으로 확인**할 수 있다.
- 완료 알림을 받으면 **샵 위치 길찾기**로 바로 수령하러 갈 수 있다.

---

## 사용자는 누구인가요?

### 고객 (배드민턴 동호인)

거트 교체를 맡기고 완료 여부가 궁금한 배드민턴 동호인. 실시간 알림으로 헛걸음 없이 라켓을 수령할 수 있다.

### 샵 사장님 (거트 전문점 운영자)

거트 작업을 접수하고 관리하는 샵 운영자. 앱으로 작업 상태를 관리하여 문의 전화를 줄이고, 회원 관리와 샵 홍보를 통해 단골 고객을 확보할 수 있다.

---

## 서비스 이용 흐름

### 고객의 흐름

```
1. 샵에 방문하여 가게 QR코드를 스캔한다. → 자동으로 해당 샵에 회원 등록
   (앱 미설치 시 QR 스캔 → 앱 다운로드 페이지로 이동 → 가입 후 자동 등록)
2. 오프라인에서 샵에 라켓을 맡긴다.
3. 사장님이 작업을 접수하면 "접수됨" 푸시 알림을 받는다.
4. 앱에서 작업 진행 현황(접수됨 → 작업중 → 완료)을 실시간으로 확인한다.
5. "완료" 푸시 알림을 받으면 샵 위치 길찾기로 라켓을 수령하러 간다.
```

### 샵 사장님의 흐름

```
1. 회원가입 시 "샵 사장님"을 선택하고, 바로 이어서 샵 정보를 등록한다. (2단계 가입)
2. 샵 고유 QR코드를 가게에 비치한다.
3. 고객이 QR을 스캔하면 자동으로 회원 등록된다. (또는 수동 등록)
4. 거트, 텐션, 라켓 정보를 입력하고 작업을 접수한다.
5. 작업 진행에 따라 상태를 변경한다. (접수됨 → 작업중 → 완료)
6. 상태 변경 즉시 고객에게 푸시 알림이 자동 전송된다.
7. 샵 페이지에 공지사항, 이벤트, 가게 재고를 올려 홍보한다.
```

---

## 핵심 차별점: 샵 QR코드 기반 간편 회원 등록

기존 샵에서는 고객 정보를 수기로 메모하거나, 별도의 관리 수단이 없는 경우가 대부분이다. 거트알림은 **샵 고유 QR코드를 가게에 비치**하여, 고객이 스캔만 하면 자동으로 해당 샵에 회원 등록되는 방식을 도입한다.

### 앱 사용 고객의 경우

고객이 가게에 비치된 QR코드를 자신의 폰으로 스캔하면, 해당 샵에 자동으로 회원 등록된다. 사장님은 별도의 등록 작업 없이 바로 작업 접수가 가능하다.

### 앱 미설치 고객의 경우

가게 QR코드를 스캔하면 앱 다운로드 페이지로 이동한다. 앱을 설치하고 가입하면 해당 샵에 자동 회원 등록된다. 또는 사장님이 수동으로 이름과 연락처만 입력하여 회원을 등록할 수 있다. 나중에 고객이 앱에 가입하면, 전화번호 기준으로 기존 회원 정보와 자동 매칭되어 이전 작업 이력까지 바로 확인할 수 있다.

## 핵심 기능

### 실시간 작업 현황 확인

고객은 앱에서 자신의 거트 작업 상태(접수됨 / 작업중 / 완료)를 실시간으로 확인할 수 있다. 각 상태 변경 시점이 타임라인으로 표시된다.

### 푸시 알림

사장님이 작업 상태를 변경하면 고객에게 즉시 푸시 알림이 전송된다. 완료 알림에는 샵 위치 바로가기가 포함되어, 탭 한 번으로 길찾기 화면으로 이동할 수 있다.

### 주변 샵 검색

네이버 지도에서 현재 위치 기준으로 주변 거트 샵을 검색할 수 있다. 지도 마커 또는 리스트 뷰로 확인 가능하며, 각 샵의 현재 작업 현황(접수 N건 / 작업중 N건)도 표시된다.

### 샵 QR코드 회원 등록

각 샵에 고유한 QR코드가 부여된다. 고객이 QR을 스캔하면 해당 샵에 자동 회원 등록된다. 앱 미설치 고객은 QR 스캔 시 앱 다운로드 페이지로 이동한다.

### 회원 관리

사장님은 회원별 방문 이력을 체계적으로 관리할 수 있다. 회원 등록은 이름과 연락처만 입력하면 되어 간편하다.

### 작업 관리 대시보드

사장님은 오늘의 작업 현황(접수 / 진행 / 완료 건수)을 한눈에 확인하고, 한 탭으로 작업 상태를 전환할 수 있다.

### 샵 페이지 (공지사항 / 이벤트 / 가게 재고)

사장님은 샵 페이지를 통해 적극적으로 홍보할 수 있다.

- **공지사항**: 샵 관련 공지사항을 게시글로 올린다.
- **이벤트**: 할인, 프로모션 등 이벤트를 기간과 함께 올린다.
- **가게 재고**: 라켓, 셔틀콕 등 상품을 이미지와 함께 등록하여 재고를 보여준다. 고객은 열람만 가능하며 구매는 불가하다. 이미지를 등록하지 않으면 기본 이미지가 표시된다.

## 주요 페이지 구성

### 공통 화면

| 페이지 | 설명 |
|--------|------|
| 스플래시 / 로그인 | 앱 진입, 역할 선택(고객 / 샵 사장님) 후 회원가입 |
| 회원가입 (고객) | 이름, 이메일, 비밀번호, 연락처 입력 → 완료 |
| 회원가입 (사장님) | 1단계: 기본 정보 → 2단계: 샵 등록 (이름, 주소, 연락처, 소개글) |

### 고객 화면

| 페이지 | 설명 |
|--------|------|
| 고객 홈 | 현재 진행 중인 내 작업 목록 (상태 표시: 접수됨/작업중/완료) |
| 작업 상세 | 거트, 텐션, 라켓, 샵 정보, 상태 타임라인, 길찾기 |
| 주변 샵 검색 | 네이버 지도에 주변 샵 마커 표시, 리스트 뷰 전환 |
| 샵 상세 | 샵 소개, 카테고리 탭 (공지사항/이벤트/가게 재고), 길찾기 |
| 작업 이력 | 과거 완료된 작업 목록 |
| 마이페이지 | 프로필 수정, 알림 설정, 로그아웃 |

### 샵 사장님 화면

| 페이지 | 설명 |
|--------|------|
| 사장님 대시보드 | 오늘의 작업 현황, 최근 작업 목록 |
| 작업 접수 | 회원 검색 → 거트/텐션/라켓 입력 → 접수 |
| 작업 관리 | 전체 작업 목록, 상태 변경 버튼 |
| 회원 관리 | 회원 목록, 검색, 회원 상세 |
| 회원 등록 | 수동 등록 폼 (이름, 연락처, 메모) |
| 샵 QR코드 | 샵 고유 QR코드 표시, 이미지 저장/공유, 인쇄용 다운로드 |
| 게시글 작성 | 공지사항/이벤트 게시글 작성, 이미지 첨부 |
| 재고 관리 | 상품 등록/수정/삭제, 이미지 등록, 재고 수량 관리 |
| 샵 설정 | 샵 정보 수정, 사장님 개인정보 수정 |

## 기술 구성 요약

| 영역 | 기술 | 역할 |
|------|------|------|
| 프론트엔드 | Flutter | iOS/Android 크로스 플랫폼 앱 |
| DB/인증 | Supabase (PostgreSQL + Auth) | 데이터 저장, 사용자 인증 |
| 푸시 알림 | Firebase Cloud Messaging (FCM) | 작업 상태 변경 시 알림 전송 |
| 알림 트리거 | Supabase Edge Function | DB 트리거 → FCM 호출 |
| 지도 | Naver Map API | 주변 샵 검색, 길찾기 연동, 주소→좌표 변환 |

## 핵심 정책

- **샵 QR코드 기반 회원 등록**: 각 샵에 고유 QR코드 부여. 고객이 스캔하면 자동 회원 등록. 앱 미설치 시 앱 다운로드 페이지로 이동
- **앱 미가입 고객 지원**: 사장님이 이름과 연락처만으로 수동 회원 등록 가능. 나중에 가입하면 전화번호로 자동 매칭
- **간편 회원 등록**: 회원 등록 시 이름과 연락처만 필수. 선호 거트/텐션/라켓 정보는 수집하지 않음
- **가입 시 역할 고정**: 고객과 샵 사장님 간 역할 전환은 불가
- **사장님 가입 시 샵 등록 필수**: 사장님 가입 2단계에서 샵 이름, 주소, 연락처를 등록. 주소 입력 시 네이버 지도 API로 자동 좌표 변환
- **작업 정보 독립 저장**: 매 작업마다 거트, 텐션, 라켓 정보를 별도 저장
- **재고 열람만 가능**: 가게 재고는 고객이 열람만 가능하며 앱 내 구매는 불가. 이미지 미등록 시 기본 이미지 표시
